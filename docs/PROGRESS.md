- **T3.3** done 2026-07-12. Dependency resolution: `resolveDependencies(store, bundlesDir, framework)` -- a generic graph resolver over `dependencies.settings`/`dependencies.roles`/`dependencies.plugins` (§6.3) that knows nothing about specific capability or plugin-type names, only the declared dependency shape. Loads manifests for every currently role-bound trusted plugin from `wanfw_bundles`, checks each declared setting against the framework doc, each declared role against current role bindings, each declared plugin dependency against the trust store, and runs a DFS cycle check over the plugin-dependency graph among role-bound plugins.
